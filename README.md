# Evolutionary Robotics Scientific Website

Бұл жоба Streamlit прототипінен GitHub Pages-ке арналған статикалық сайтқа айналдырылды.

## Функциялар
- Wikipedia-style ғылыми бөлімдер
- Іздеу жүйесі
- Dark / Light режим
- Chart.js интерактивті диаграммалары
- Fitness function калькуляторы
- GA simulation slider
- Ғылыми кесте
- Робот түрлері
- Терминологиялық сөздік
- Галерея
- Әдебиеттер

## GitHub Pages арқылы жариялау
1. Репозиторий ашыңыз.
2. Осы файлдарды толық жүктеңіз.
3. Settings → Pages → Deploy from branch таңдаңыз.
4. Branch: main, Folder: /root.
5. Save басыңыз.
